# 政解 - 西红门民生政策助手

“政解”面向北京市大兴区西红门镇基层民生服务场景，围绕“人找政策”和“政策找人”两条主线，将政策查询、居民档案、资格初筛、材料准备和代办任务连接起来。

系统内的居民姓名和档案均为虚构演示数据。匹配结果只用于工作人员初步核查，不构成正式审批结论，最终以相关经办部门审核为准。

## 立即体验

[打开“政解”公开演示](https://shenicest-2.vercel.app)

电脑和手机都可以直接打开这个网址，无需安装软件、登录账号、配置模型 Key 或运行任何命令。

请把上面的公开网址发给体验者，不要发送 `http://localhost:3000`。`localhost` 只代表当前设备本身：它只能在启动了本地服务的那台电脑上访问，关闭服务后也会失效；在手机上打开 `localhost`，访问的是手机自己，不是电脑上的项目。

## 电脑端怎么使用

电脑浏览器打开公开网址后，会自动进入完整工作台。顶部可以切换“政策知识库”“居民档案”和“代办台账”。

### 政策找人

1. 进入“政策知识库”，搜索或选择一项政策。
2. 点击“查看详情”，阅读适用对象、待遇标准、所需材料和官方出处。
3. 点击“匹配本村居民”，查看高度匹配、待核实和暂不匹配的居民。
4. 阅读每位居民的匹配原因、排除原因和缺失信息，再点击“查看居民档案”继续核查。

### 人找政策

1. 进入“居民档案”，搜索或选择一名居民。
2. 打开居民详情，查看该居民可能匹配和需要继续核实的政策。
3. 根据页面提示补充现场信息，并查看政策依据和材料清单。
4. 需要继续跟进时，点击“加入代办台账”。

### 跟进代办事项

进入“代办台账”，可以查看已加入的办理事项，并更新材料收集和办理状态。

## 手机端怎么使用

用手机浏览器打开同一个公开网址，首页会显示四个精简入口：

- **人找政策**：选择居民，从居民档案查看可能符合的政策。
- **政策找人**：选择政策，查找可能符合或需要继续核实的居民。
- **快速搜索与现场补录**：搜索居民、进入档案，并补充现场核实的信息。
- **今日任务**：查看需要继续核实和跟进的代办事项。

手机端适合现场快速查询、补录和跟进；需要集中浏览大量政策或居民信息时，建议使用电脑端。

## 数据保存说明

- 现场补录和代办状态保存在当前设备的当前浏览器中。
- 电脑和手机、不同浏览器之间不会自动同步记录。
- 使用无痕模式、清除浏览器数据或更换设备后，本地记录可能消失。
- 请勿在演示系统中录入真实身份证号、电话号码、住址等个人敏感信息。

## 核心功能

- 政策列表、分类检索、政策详情和官方出处；
- 根据政策查找可能符合或需要核实的居民；
- 根据居民档案查找可能匹配的政策；
- 展示匹配原因、排除原因和缺失信息；
- 查看材料清单、现场补录信息并添加代办任务；
- 桌面端完整工作台与手机端精简办事入口。

## 技术架构

- Next.js 16 App Router、React 19 和 TypeScript；
- 原生 CSS 与 CSS Modules 响应式界面；
- 本地政策知识库与虚构居民演示数据；
- 确定性政策匹配脚本负责资格条件计算；
- 大语言模型辅助政策结构化和字段语义对齐，不直接决定居民是否符合资格；
- 通过 Vercel 提供电脑端和手机端公开访问。
